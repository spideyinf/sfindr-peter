Create a reactive form with the following fields:

  • name: text input
  • type: dropdown with values: 'Car', 'Bike'

When a value in the type dropdown is selected, another field should be added based on the type. If it's "car", add a text input with the name brand. If it's "bike", add a text input with the name color. Be sure to remove the other input, when the type changes, so that its value is not part of the result object.

All fields are required.

A submit button should call the injected service with the value of the form, but should only be enabled if the form is valid.
